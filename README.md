# Ourshelves

A localized, zero-cost book sharing and exchange platform that facilitates free lending and exchange of physical books within local communities.

## Project Structure

```
ourshelves/
├── client/          # React + Vite frontend
├── server/          # Express.js backend
└── package.json     # Monorepo root
```

## Getting Started

### Prerequisites
- Node.js 18+
- npm 9+

### Installation

```bash
# Install all dependencies
npm install

# Start both client and server in development mode
npm run dev
```

### Development

- **Client**: http://localhost:5173
- **Server**: http://localhost:3000

## Features (Phase 1 MVP)

- User registration and authentication
- Book cataloging with ISBN lookup
- Basic search functionality
- Lending/exchange transaction workflow
- User ratings and reviews
